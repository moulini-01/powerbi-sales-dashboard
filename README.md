# 📊 Sales Performance Dashboard (Power BI)

## 🚀 Project Overview
This project is an interactive Power BI dashboard that analyzes sales performance across different regions, categories, and customer segments.

## 📌 Features
- Region-wise analysis (Central, East, South, West)
- Category insights (Furniture, Office Supplies, Technology)
- KPI metrics (Revenue, Profit, Quantity, Discount)
- Year-wise sales trend analysis
- Interactive filters (Region, Segment, Ship Mode)

## 🛠️ Tools Used
- Power BI
- Excel
- Data Cleaning & Transformation
- Basic DAX

## 📈 Insights
- Technology category contributes highest revenue
- Sales show growth trend over time
- Regional comparison helps identify performance differences
